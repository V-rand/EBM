Search PubMed biomedical literature (MEDLINE + PubMed Central). Core tool for EBM research with Clinical Queries and article type filters.

### EBM-specific parameters
- `article_type="rct"` — Randomized controlled trials only
- `article_type="systematic_review"` — Systematic reviews only
- `article_type="meta_analysis"` — Meta-analyses only
- `article_type="guideline"` — Clinical practice guidelines
- `clinical_query="therapy"` — Therapy clinical queries (validated PubMed search strategies)
- `clinical_query="diagnosis"` — Diagnosis queries
- `clinical_query="prognosis"` — Prognosis queries
- `clinical_query="etiology"` — Etiology/harm queries

Returns `{total_count, results:[{pmid,title,authors,journal,pubdate,doi,url}], count, filters_applied}`. Max 20 results.
