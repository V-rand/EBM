# Evidence Appraisal — 证据质量评估

## 何时使用

当需要评估临床研究证据的质量、为报告中的证据等级做评级时使用。适用于系统评价中的证据质量评价、临床指南的证据分级、以及研究结论的可信度判断。

## GRADE 证据质量分级

GRADE（Grading of Recommendations Assessment, Development and Evaluation）是目前使用最广泛的证据质量评估系统。

### 起始等级

| 研究设计 | 起始等级 |
|----------|---------|
| RCT（随机对照试验） | **高质量 (High)** |
| 观察性研究（队列/病例对照） | **低质量 (Low)** |
| 病例系列/个案报告 | **极低质量 (Very Low)** |

### 降级因素（-1 或 -2）

| 因素 | 说明 | 评估方法 |
|------|------|---------|
| **偏倚风险 (Risk of bias)** | 随机化不充分、未盲法、失访多 | 用 RoB 2.0（RCT）或 ROBINS-I（观察性） |
| **不一致性 (Inconsistency)** | 异质性大（I² > 50%）、结果方向不一致 | 检查森林图、亚组分析 |
| **间接性 (Indirectness)** | 患者/干预/结局与问题不匹配 | 检查 PICO 一致性 |
| **不精确性 (Imprecision)** | 置信区间宽、样本量小 | 检查 95% CI 和 OIS |
| **发表偏倚 (Publication bias)** | 阴性结果未发表 | 漏斗图对称性 |

### 升级因素（仅观察性研究，+1 或 +2）

| 因素 | 条件 |
|------|------|
| **效应量大** | RR > 2 或 < 0.5（+1）；RR > 5 或 < 0.2（+2） |
| **剂量-反应关系** | 存在梯度效应 |
| **混杂偏倚减弱效应** | 实际效应可能更大 |

### 最终分级

| 等级 | 含义 |
|------|------|
| **高质量 (High)** ⊕⊕⊕⊕ | 进一步研究几乎不可能改变效应估计的信度 |
| **中等质量 (Moderate)** ⊕⊕⊕○ | 进一步研究可能对效应估计产生重要影响，可能改变估计值 |
| **低质量 (Low)** ⊕⊕○○ | 进一步研究很可能对效应估计产生重要影响，很可能改变估计值 |
| **极低质量 (Very Low)** ⊕○○○ | 对效应估计的任何信度都非常不确定 |

## Oxford CEBM 证据等级

适用于临床问题导向的证据分级：

| 等级 | Therapy/Prevention | Diagnosis | Prognosis |
|------|-------------------|-----------|-----------|
| **1a** | 同质性RCT的系统评价 | 同质性诊断性研究的系统评价 | 同质性队列研究的系统评价 |
| **1b** | 单个RCT（窄CI） | 单个诊断性研究 | 单个队列研究（随访>80%） |
| **1c** | 全或无效应 | 全/高特异性诊断 | 全或无预后 |
| **2a** | 同质性队列研究的系统评价 | 同质性诊断性研究的系统评价 | 回顾性队列或未治疗对照的RCT |
| **2b** | 单个队列研究（含低质量RCT） | 单个诊断性研究 | 回顾性队列 |
| **3a** | 同质性病例对照研究的系统评价 | 同质性诊断性研究的系统评价 | — |
| **3b** | 单个病例对照研究 | 非连续诊断研究 | — |
| **4** | 病例系列/低质量队列或病例对照 | 病例对照研究 | 病例系列 |
| **5** | 专家意见（无严格评价） | 专家意见 | 专家意见 |

## 证据链评估（EBM 特有）

在循证医学中，GRADE 不是对单篇文献的孤立评估，而是对整条证据链**逐跳评估**：

```
临床推荐
    ↑ 这一跳的 GRADE？—— 指南本身的方法学质量
系统评价/指南
    ↑ 这一跳的 GRADE？—— 纳入研究的质量与一致性
原始研究 (RCT/观察性)
    ↑ 这一跳的 GRADE？—— 单项研究的设计与偏倚风险
```

**逐跳评估规则：**
- 整条链的置信度 ≤ 最弱一跳的 GRADE
- 指南 Ⅰ 类推荐 ≠ 证据高质量（指南可能基于低质量证据做 strong recommendation）
- 系统评价的质量取决于其纳入的 RCT 质量：即使 meta 分析方法学完美，纳入的 RCT 偏倚风险高时，该跳仍降级

**链路完整性判定：**

| 链路状态 | GRADE 上限 | 报告标注 |
|----------|-----------|---------|
| 指南 ← 系统评价 ← 多项高质量 RCT | High | "证据链完整，GRADE 高质量" |
| 指南 ← 系统评价 ← 少量/偏倚 RCT | Moderate-Low | "系统评价方法学良好但纳入 RCT 质量参差" |
| 系统评价 ← 多项 RCT（无指南）| Moderate | "链缺少指南层——已整合证据但未形成临床推荐" |
| 单项 RCT（无系统评价）| Low | "单项 RCT，缺少证据整合层" |
| 仅观察性研究 | Very Low | "链在 RCT 层断开——仅有关联证据" |

## 评估流程

### 1. 确定研究类型以及在证据链中的位置
先阅读文献摘要/全文，确定设计类型，**明确该研究在证据链中的层级**（单项研究 / 系统评价 / 指南）。

### 2. 关键方法学评估

**对于 RCT**（使用 Cochrane RoB 2.0）：
- 随机化过程是否充分？
- 是否分配隐藏？
- 是否对参与者/人员/结局评估者设盲？
- 结局数据是否完整？
- 是否有选择性报告？

**对于观察性研究**（使用 ROBINS-I）：
- 混杂偏倚控制？
- 选择偏倚？
- 干预分类偏倚？
- 偏离预期干预？
- 缺失数据？
- 结局测量偏倚？
- 选择性报告？

### 3. 记录评估

将证据评估结果写入 `research_state`：

```
research_state(content="
## 证据评估
### 研究: EMPA-REG OUTCOME (Zinman et al, 2015)
- 设计: RCT, 多中心, 双盲, 安慰剂对照
- 样本量: n=7,020
- 随访: 中位3.1年

### GRADE 评估
- 起始: High (RCT)
- 降级: 0（方法学质量高）
- 升级: 0
- 最终: ⊕⊕⊕⊕ 高质量

### 关键发现
- 主要结局: 3-point MACE HR 0.86 (0.74-0.99), p=0.04
- NNT: 39（3年）
- 质量评价: 方法学质量高，结论可靠
")
```

### 4. 在报告中呈现

引用时注明证据等级：
> 「SGLT-2抑制剂可降低2型糖尿病患者MACE风险 (HR 0.86, 95%CI 0.74-0.99) [1]」
> — *证据等级: GRADE 高质量 ⊕⊕⊕⊕, Oxford 1b*

## 常见陷阱

- ❌ 把低质量 RCT 等同于高质量证据
- ❌ 不做降级/升级判断就直接给 GRADE 等级
- ❌ 忽略异质性和间接性
- ❌ 把动物实验/体外研究的证据等同于临床证据
- ❌ 不区分统计学显著性和临床重要性
