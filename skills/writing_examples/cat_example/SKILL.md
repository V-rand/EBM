# CAT 写作范例——Cochrane Plain Language Summary

## 这是什么

Cochrane Plain Language Summary 是目前最好的临床快评范例——免费访问、由 Cochrane 方法学家撰写、遵循"问题→答案→证据→结论"结构。适合作为 CAT 写作参考。

## 在写 CAT 之前

用 `web_read` 打开以下 1-2 篇，花 2 分钟看结构——不要精读内容，重点看怎么写：

### 推荐范例（免费、可直连访问）

**范例 1：Cochrane 临床快评**
```
web_read(url="https://www.cochrane.org/CD014717/mobile-health-technologies-improve-walking-distance-intermittent-claudication")
```
这是一篇关于移动健康技术改善间歇性跛行患者步行距离的 Cochrane 系统评价的通俗摘要。

**范例 2：CDC 临床实践指南推荐**
```
web_read(url="https://www.cdc.gov/mmwr/volumes/73/rr/rr7301a1.htm")
```
CDC MMWR 推荐页面——展示推荐如何结构化呈现、证据强度如何附着。

### 看什么

打开页面后，只看这几样东西：
1. **怎么开头**——第一段是什么？是直接给答案还是先讲背景？
2. **关键数字在哪**——效应量、CI 是嵌在段落里还是单独列？
3. **证据质量怎么表达**——用 GRADE 符号（High/Moderate/Low/Very Low）还是用自然语言（"high certainty"）？
4. **多长**——整篇多少段？一段多少句？
5. **结论怎么收尾**——是重申推荐还是指出局限性？

### 看完后写

看完了就开始写你自己的 CAT。**不要抄范例的内容——学它的写法结构，不是抄它的词。**
